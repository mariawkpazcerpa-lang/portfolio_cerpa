Demo: Base de Datos de Clientes

Registro Inicial: Muestra el proceso paso a paso para ingresar los datos de nuevos clientes (Rosa y Pedro) a través del menú interactivo.

Visualización de la Base: Se observa el menú principal y la ejecución de la opción "Visualizar Base de Datos", mostrando la lista completa de clientes registrados hasta el momento.

Búsqueda Eficiente: Demuestra la funcionalidad de búsqueda de clientes. En este caso, se utiliza la opción de búsqueda por ID para recuperar un registro específico (`ID: 3).

Actualización de Clientes: Muestra el proceso completo para modificar un registro. Se actualiza el nombre de 'Rosa Pérez' a 'Rosalía', confirmando el cambio con el mensaje de éxito..

Confirmación de Eliminación: Ilustra los pasos para eliminar un cliente. Se pide una confirmación de la acción y se cancela la eliminación, regresando al menú principal.

Control Total (CRUD): Muestra la eliminación definitiva de un cliente (`ID: 3). Tambien muestra la salida del menú.
