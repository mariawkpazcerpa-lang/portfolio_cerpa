Demo: Sistema de Inventario de Productos

Alta de Inventario: Muestra el proceso paso a paso para registrar nuevos productos (Fideos y Huevos) con todos sus detalles (nombre, precio, cantidad y categoría).

Actualización de Stock: Se inicia el proceso de actualización de productos tras visualizar la lista. Específicamente, se ingresa la nueva Cantidad de unidades para el producto seleccionado..

Gestión de Baja Cantidad: Se observa una alerta de stock al intentar actualizar. Posteriormente, se demuestra la función de Eliminar un Producto (ID: 2), pidiendo la confirmación y completando el borrado..

Control Post-Eliminación: Muestra el inventario final con el producto eliminado. Luego, se visualiza la lista de productos restante (solo ID: 1) antes de proceder con la opción de Salir del sistema.
